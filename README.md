# Keyboard Drummer

Browser rhythm game inspired by Guitar Hero, built for keyboard drumming.

## Features
- Falling-note rhythm gameplay
- Lefty/righty key mapping
- Jam mode and recording mode
- Multiple drum kits (Studio Maple, Arena Rock, Electronic)
- Preloaded songs plus replayable recorded takes

## Run locally
```bash
./start_game.sh
```

Then open the URL printed by the script.
